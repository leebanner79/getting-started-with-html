The web is mainly made up of HTML documents.
This stands for Hypertext Markup Language
HTML is the language which tells a computer how something should look!

HTML uses lots of different tags to make text content appear in a certain way.
For example the word bold with no tag looks like any other word until a bold tag is used.

When someone wants to access the web they do so through a browser such as google chrome. A broweser converts the HTML into what we can see on the computer screen.
The HTML files are sent from servers via the internet using a protocol called HTTP
A server is just a computer like you have at home but normally kept in a datacentre
